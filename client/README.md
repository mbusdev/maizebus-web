# Maize Bus - Web
> Dynamic site for MaizeBus built with the MERN stack

[![NPM Version][npm-image]][npm-url]

![](header.png)

## Installation

```sh
npm install -y
```

## Development setup

To run the site for development, enter this into your terminal:

```sh
npm run dev
```

and open the localhost link given to you
